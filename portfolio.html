2-Player Online Chess Game

Full-Stack Developer

Firebase, jQuery, CSS Grid, SweetAlert.js

Uses Google Firebase to store each turn, whose turn it is, and a few other things to enable a fully-functional online chess game. Allows each player (not spectators) to undo the last turn (with opponent's consent), or start a new game if they desire. Uses SweetAlert to notify users if it is their turn, they are in check, or if their opponent wants to undo.

Front-End (React) Developer

React, Marked.js, create-react-app
